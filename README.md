# 📋 월간세미나 포스터 생성기

소부장기술융합포럼 월간세미나 홍보 포스터 자동 생성 도구입니다.  
GitHub 연동을 통해 매월 제작한 포스터를 기록하고 팀원들과 공유할 수 있습니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| 📝 **동적 편집** | 제목·일시·장소·프로그램 실시간 수정 및 미리보기 |
| � **GitHub 히스토리** | 매달 생성한 데이터를 GitHub에 JSON 파일로 영구 보존 |
| � **아카이브 복구** | 과거 보관했던 데이터를 클릭 한 번으로 다시 불러오기 |
| � **이미지 내보내기** | 고해상도 PNG 파일 저장 및 클립보드 즉시 복사 |

---

## � 시작하기 (GitHub 연동 방법)

팀원들과 공유하여 사용하려면 최초 1회 GitHub 연동이 필요합니다.

1.  **GitHub 저장소 생성**: GitHub에서 포스터 데이터를 저장할 전용 리포지토리를 만듭니다.
2.  **토큰(PAT) 발급**:
    *   `Settings` > `Developer settings` > `Personal access tokens` > `Fine-grained tokens`
    *   새 토큰 생성 시 해당 저장소에 대해 **Contents: Read and write** 권한을 부여합니다.
3.  **앱 설정**: 웹 앱 상단의 **[⚙️ GitHub 설정]** 버튼을 눌러 정보(ID, 저장소명, 토큰)를 입력합니다.

---

## � 사용 방법

1.  **내용 입력**: 왼쪽 패널에서 해당 월의 정보를 입력합니다.
2.  **포스터 생성**: `▶ 포스터 생성` 버튼을 눌러 디자인을 확인합니다.
3.  **아카이브 보관**: `🐙 GitHub에 월간 포스터 보관` 버튼을 누르면 GitHub에 기록됩니다.
4.  **과거 기록 열람**: 상단 **[📁 아카이브 히스토리]** 버튼을 통해 과거 데이터를 조회하고 복구합니다.

`index.html` 소스 코드를 직접 수정하여 브랜드 톤을 변경할 수 있습니다.
