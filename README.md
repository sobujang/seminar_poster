# 📋 월간세미나 포스터 템플릿

소부장기술융합포럼 월간세미나 홍보 포스터 자동 생성 도구입니다.  
매달 내용만 입력하면 동일한 디자인의 포스터를 즉시 만들 수 있습니다.

---

## 🔗 바로 사용하기

👉 **[포스터 만들러 가기](https://[내아이디].github.io/seminar-poster/)**

> GitHub Pages 주소로 교체해주세요

---

## ✨ 기능

| 기능 | 설명 |
|---|---|
| 📝 내용 편집 | 왼쪽 폼에서 제목·일시·장소·프로그램 입력 |
| ➕ 프로그램 추가 | 발표 세션 수에 맞게 자유롭게 추가/삭제 |
| 🖼 이미지 저장 | PNG 파일로 자동 다운로드 |
| 📋 클립보드 복사 | 카카오톡·슬랙·메일에 바로 붙여넣기 |
| 🖨 인쇄/PDF | 브라우저 인쇄로 PDF 저장 |

---

## 🖥 사용 방법

1. 위 링크로 접속
2. **왼쪽 폼**에서 해당 월 내용 수정
   - 메인 타이틀 (예: `2026년 5월 / 월간세미나`)
   - 일시·장소
   - 프로그램 항목 (추가/삭제 가능)
3. **▶ 포스터 생성** 버튼 클릭
4. 오른쪽 미리보기 확인 후 저장

---

## 🗂 매달 변경하는 항목

| 프로그램 내용·발표자 | ✅ 매달 변경 |

## 🛠 디자인 수정이 필요할 때

`index.html` 파일에서 아래 항목을 찾아 수정합니다.

| 항목 | 위치 |
|---|---|
| 메인 색상 변경 | `#2d2de0` 검색 후 원하는 색상 코드로 교체 |
| 기본 기관명 변경 | `value="소부장기술융합포럼"` |
| 기본 장소 변경 | `value="티로보틱스 (경기도..."` |
| 기본 프로그램 변경 | `let programs = [...]` 배열 수정 |

수정 후 GitHub에 파일을 다시 업로드하면 자동 반영됩니다.

## 📌 참고

- **권장 브라우저**: Chrome (클립보드 복사 최적화)
- **이미지 해상도**: 2배 고해상도 PNG로 저장됨
- 로컬 파일(`file://`)로 열면 클립보드 복사가 제한될 수 있습니다. GitHub Pages 링크를 이용해주세요.
